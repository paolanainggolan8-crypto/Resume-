Saya memiliki kemampuan dalam menggunakan HTML, CSS, JavaScript, dan PHP untuk membangun website yang modern, responsif, serta mudah digunakan oleh pengguna. Saya senang mempelajari teknologi baru dan selalu berusaha meningkatkan kemampuan agar dapat mengikuti perkembangan dunia digital yang terus berkembang. Oleh karena itu, saya selalu berusaha menciptakan website yang tidak hanya memiliki tampilan yang menarik, tetapi juga fungsional, aman, dan sesuai dengan kebutuhan pengguna.

Pengalaman

Frontend Developer

2023 - Sekarang


Sebagai Frontend Developer, saya bertanggung jawab dalam merancang dan mengembangkan tampilan website yang menarik, responsif, dan mudah digunakan oleh pengguna. Dalam pekerjaan ini, saya menggunakan berbagai teknologi web seperti HTML, CSS, dan JavaScript untuk membangun antarmuka pengguna (User Interface) yang modern dan interaktif. Saya terlibat dalam proses perencanaan desain, pengembangan fitur, pengujian tampilan website, hingga proses optimasi agar website dapat berjalan dengan baik pada berbagai perangkat seperti komputer, tablet, maupun smartphone. Selain itu, saya juga memastikan bahwa setiap elemen website dapat diakses dengan mudah oleh pengguna serta memiliki performa yang optimal. Beberapa tugas yang telah saya kerjakan meliputi: Membuat desain halaman website yang responsif dan user-friendly. Mengembangkan fitur interaktif menggunakan JavaScript. Mengoptimalkan tampilan website agar kompatibel di berbagai browser. Melakukan perbaikan bug dan peningkatan performa website. Bekerja sama dalam pengembangan sistem berbasis web sesuai kebutuhan pengguna. Mengelola dan memperbarui konten website secara berkala. Meningkatkan pengalaman pengguna (User Experience) melalui desain dan navigasi yang lebih baik. Melalui pengalaman ini, saya berhasil meningkatkan kemampuan dalam pemrograman web, pemecahan masalah, kerja sama tim, manajemen waktu, serta memahami kebutuhan pengguna dalam pengembangan sebuah website yang berkualitas.
